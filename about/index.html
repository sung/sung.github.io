How this site was made

Posted at — Aug 5, 2019

In case you’re curious, I used blogdown which uses Hugo internally. This theme is based on ezhil (with some modification). I like RStudio, but prefer Nvim-R which is great if you like Vim & terminal.

The source code (of blogdown & hugo bits) is avaible here and the static contents are deployed on GitHub. You may find this useful, if you’d like to make your github page as a submodle of your Hugo codes.
